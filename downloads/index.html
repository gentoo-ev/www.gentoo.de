Gentoo zum Ausprobieren: Die Gentoo-LiveDVD

Die Gentoo-LiveDVD (oder auch Hybrid ISO, da sie auch für USB-Sticks geeignet ist) ist ein schneller Weg, Gentoo zu testen. Sie bietet eine breite Auswahl an Paketen unserer Distribution ohne gesonderte Installation.

Wir stellen zwei Versionen für amd64 zur Verfügung: Minimal Installation CD und LiveGUI USB Image. (Details zu den Architekturen finden sich im Gentoo-Handbuch.)

amd64 Minimal Installation CD amd64 LiveGUI USB Image

Gentoo zum Installieren: Stages und Minimal LiveCD

Wenn Gentoo auf die Platte (oder SSD) soll, dann werden weitere Dateien benötigt.

Wann welche Dateien genau benötigt werden, wird im Installationshandbuch beschrieben. Diese finden sich alle auf der Downloadseite auf gentoo.org, auf die rechts verwiesen wird.
